# Word Stock

`words.csv`を使った4択式の英単語テストサイトです。

## 起動方法

ブラウザの制約によりCSVは`file://`から読み込めないため、このフォルダでローカルサーバーを起動します。

```powershell
python -m http.server 8000
```

起動後、ブラウザで <http://localhost:8000> を開いてください。

## 機能

- 英語から日本語、日本語から英語の出題切替
- その場で採点する4択モード
- 全問出題後に答えを確認して自己採点するテストモード
- 答え合わせ中の採点修正
- 累計正答率、間違えた問題、単語別の間違い回数を保存
- 間違えた問題を回数・復習状態・出題方向・テストで絞り込み、選択して再テスト
- 完了テストを日時・方向・モード・問題数・範囲・短縮IDで識別
- 再テストで正解した問題を「克服済み」として管理
- 自分で「要注意」タグを付け、絞り込み・再テスト
- IDによる出題範囲指定とカテゴリプリセット
- 10問、20問、50問、範囲内すべてから問題数を選択
- キーボード操作（数字キー、`Enter`、左右矢印キー）
- 結果表示、間違えた単語の復習、ベストスコア保存
